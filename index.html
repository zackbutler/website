Image Restoration (In Progess)

The Image Restoration project is a machine learning program based around restoring damaged photos. There are two portions of the project, the first being the damager, which creates realistic blemishes on a large amount of pictures. Because there isn't a dataset of beat-up photographs before and after the damage, we've created a number of methods to simulate common photo deformities in pictures, such as folds, liquid stains, color distortion, etc. Implementing our now usable dataset, we are planning on using the second portion of our project, the restorer, to train itself on our data and be able to realistically recreate the photos as if they were unharmed, using partial convolution (somewhat similar to regular CNNs). After training, we will be able to implement this with actual damaged photographs. Find more information about our project here.

Automated Algorithm Design

Automated Algorithm Design research is a research design team I am working on to create and optimize hybrid human/machine algorithms. The goal of the team is to develop a framework that starts with human algorithms, and then uses automated techniques (specifically multi-objective genetic programming) to create better, outperforming algorithms that improve upon the previous ones. Rather than implementing standard genetic programming to evolve our algorithm, we are using higher-level input features to allow for more advanced basis functions. The combination of this with a standard genetic algorithm enables our program to create human-readable algorithms across a number of desired objectives.
